# freddie-java

> Personal collection of data structure & algorithm implementations, NeetCode problem solutions, and generated problems for university practice.

## Structure

```
implementations/
  data_structures/
    linked_lists/
      singly_linked_list/       # README.md + SinglyLinkedListMain.java
    trees/
      binary_search_tree/       # README.md + BinarySearchTreeMain.java
  algorithms/
    sorting/
      merge_sort/               # README.md + MergeSortMain.java
      quick_sort/               # README.md + QuickSortMain.java
      insertion_sort/           # README.md + InsertionSortMain.java
    bfs_dfs/
      depth_first_search/       # README.md + DepthFirstSearchMain.java

problems/                       # Problem sheets and solutions
```

## Java Style Note

All classes and `main` methods are written together in a single `.java` file rather than split into separate files per class. This intentionally deviates from Java best practice to keep things simple for practice purposes.
